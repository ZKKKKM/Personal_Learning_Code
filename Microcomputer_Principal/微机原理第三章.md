## 内存/存储 （Memory）

内存就是暂时再计算机内部存储的指令以及数据。
#### 内部存储

内部存储包括了ROM以及RAM 它们的的特点在于**随机访问，访问速度快，以及具有小容量的存储。**

#### 外部存储

外部存存储包含了磁盘，光盘 （optical disk，半导体存储器（semiconductor memory），它的特点是**顺序访问/块访问，低访问速度，大存储容量**

#### 存储单元的地址和内容

内存中是包含这一些存储单元的，每个存储单元包含8bit, 为了区分每一个存储单元，我们就将存储单元中的数字称作存储单元的地址（The address of the location）

#### 存储容量和存储空间

- 存储容量
	存储容量（Memory Capacity） 代表的是计算机中**实际安装了多少内存条**，即实际有多少个位（bits）或字节（bytes）可以用来存数据。
	
	例如：如果**一台微机有两个 $128\text{MB}$ 的 SDRAM，那么它的 Memory Capacity 就是 $256\text{MB}$**（要记住 考试可能会考）
	
	决定因素：取决于你购买并插在主板上的内存芯片（如 SDRAM）的总大小
- 存储空间
	定义：存储空间是指 CPU **能够寻找地址的最大范围**，也就是 CPU 理论上能管理多少个房间。
	
	决定因素：它完全取决于 **地址总线（Address Bus）的宽度**。


|      | Memory Space (空间)      | Memory Capacity (容量)    |
| ---- | ---------------------- | ----------------------- |
| 关注点  | **最大范围** (Potential）   | **实际总量** (Actual Store) |
| 硬件决定 | CPU 地址总线宽度             | 内存条/芯片的大小               |
| 计算方式 | **$2^{\text{地址线位数}}$** | 芯片数量 $\times$ 单个芯片容量    |
| 关系比喻 | 门牌号的范围（0 到 99）         | 实际盖好的房子（一共 10 间）        |


#### 存储操作

1. 读：把存储单元里的 **Contents**（内容）传给 CPU。就像从房间里把人请出来，但房间里的人还在（内容不变）。
2. 写：把信息从 CPU 传到 **Locations**（位置）。这会“Cover the original data”（覆盖原数据），就像换了一批新的房客入住。
3. ==刷新： 这个只会在**DRAM （Dynamic Random Access Memory）** 中出现==


#### 存储/内存的分类

1. RAM (Random Access Memory)
	- 既能读取又能写入
	-  数据的易失性，临时存储
	1. SRAM (Static Random Access Memory) —— 静态随机存取存储器
		- 存储原理： 利用双稳态触发器（Flip-Flop）来存储信息。
		- 速度快
		- 不需要刷新
	2. DRAM (Dynamic Random Access Memory) —— 动态随机存取存储器
		- 存储原理： 利用电容器（Capacitor）存储电荷来表示数据。
		- **必须刷新 (Refresh)：** 由于电容器会自动漏电，因此必须每隔一段时间（毫秒级）就重新给它充电，否则数据会丢失。
		- **速度较慢：** 因为存在充放电过程和等待刷新的时间，速度比 SRAM 慢。
2. ROM (Read Only Memory)
	- 仅限阅读
	- 非易失性，永久性或半永久性,数据存储
